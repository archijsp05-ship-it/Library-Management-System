# Library-Management-System
A Java console application demonstrating Object-Oriented Programming concepts by implementing a Library Management System with book issue, return, search, and inventory management functionalities

## Features
- Add Books
- Display Available Books
- Issue Books
- Return Books
- Search Books by Title
- Menu-Driven Interface

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner

## How to Run
1. Clone the repository.
2. Open the project in IntelliJ IDEA or any Java IDE.
3. Compile and run `LibraryManagementSystem.java`.

## Concepts Used
- Classes and Objects
- Encapsulation
- Collections (ArrayList)
- Conditional Statements
- Loops
- User Input Handling
